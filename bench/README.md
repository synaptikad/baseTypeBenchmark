# Scripts de benchmark

Les scripts de ce dossier orchestreront les campagnes de tests : préparation des bases, exécution des requêtes, collecte des métriques et export des résultats. Ils devront gérer les répétitions, le warmup et la journalisation structurée.

Aucun script n'est présent pour l'instant ; ce dossier sera complété dès que la génération et le chargement des données seront opérationnels.
