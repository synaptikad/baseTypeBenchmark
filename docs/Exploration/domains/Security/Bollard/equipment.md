# Bollard (Borne Escamotable)

## Identifiant
- **Code** : BOLLARD
- **Haystack** : bollard
- **Brick** : brick:Retractable_Bollard

## Description
Borne de sécurité cylindrique rétractable, motorisée, émergeant du sol pour bloquer l'accès véhicule et se rétractant pour autoriser le passage. Utilisée pour protection périmétrique, zones piétonnes temporaires, ou contrôle d'accès haute sécurité. Peut résister à des impacts de véhicules à vitesse élevée (versions anti-bélier).

## Fonction
Protection physique contre intrusion véhicule, contrôle accès sélectif de véhicules, sécurisation zones piétonnes, protection anti-terroriste (sites sensibles), gestion automatique ou manuelle de l'accès, intégration avec systèmes de contrôle d'accès.

## Variantes Courantes
- **Borne standard** : Protection modérée, usage commercial
- **Borne anti-bélier (crash-rated)** : Résiste impact véhicule lourd haute vitesse (K4, K8, K12)
- **Borne semi-automatique** : Déverrouillage motorisé, manipulation manuelle
- **Borne automatique** : Escamotage entièrement motorisé
- **Borne hydraulique** : Système hydraulique, très robuste
- **Borne électromécanique** : Motorisation électrique, standard

## Caractéristiques Techniques Typiques
- Diamètre : 150-300mm
- Hauteur émergée : 600-1000mm
- Temps escamotage : 3-8 secondes
- Résistance impact : Jusqu'à 7.5 tonnes à 80 km/h (K12)
- Motorisation : Hydraulique ou électromécanique
- Alimentation : 230V AC, 24V DC
- Cycles : 1000-3000 cycles/jour selon modèle
- Communication : RS-485, TCP/IP, relais
- Commande : Badge reader, push button, télécommande, BMS
- Matériau : Acier inox 304/316, acier peint
- Éclairage : LED intégré (optionnel)
- Protection : IP67-IP68 (immersion)
- Batterie secours : Optionnelle

## Localisation Typique
- Périmètres de bâtiments sensibles (ambassades, ministères)
- Entrées de zones piétonnes
- Places et esplanades
- Accès cours intérieures
- Sites industriels critiques
- Data centers
- Infrastructures critiques
- Accès pompiers

- Moyen (15 étages) : 2-8 bornes (accès/périmètre)
- Grand (30+ étages) : 5-20 bornes (multiples accès sécurisés)
