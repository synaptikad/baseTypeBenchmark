# Speed Gate

## Identifiant
- **Code** : SPEED_GATE
- **Haystack** : speed-gate
- **Brick** : brick:Speed_Gate

## Description
Barrière de contrôle d'accès piéton haut de gamme avec portes coulissantes ou battantes ultra-rapides (verre, plexiglas, acier). Design moderne et esthétique pour halls prestigieux. Temps d'ouverture/fermeture très rapide (0.6-1.5s) permettant haut débit de passages. Détection avancée anti-tailgating et anti-piggybacking.

## Fonction
Contrôle d'accès haute sécurité et haut débit, détection optique multi-faisceaux pour anti-tailgating, gestion bidirectionnelle, intégration lecteurs multiples, modes de fonctionnement variés, interface élégante pour environnements corporatifs.

## Variantes Courantes
- **Panneaux coulissants** : Mouvement latéral horizontal
- **Panneaux battants** : Ouverture en battant (swing)
- **Panneaux repliables** : Verre se replie en accordéon
- **Speed lane** : Couloir complet avec multiples capteurs
- **Modèle PMR** : Passage élargi accessible
- **Version premium** : Finitions luxe (verre, inox miroir)

## Caractéristiques Techniques Typiques
- Temps d'ouverture : 0.6-1.5 secondes
- Débit : 30-60 passages/minute
- Largeur passage : 550-900mm (jusqu'à 1200mm PMR)
- Hauteur barrières : 900-1800mm
- Détection : Barrières infrarouges multicouches (10-30 faisceaux)
- Anti-tailgating : Détection temps réel avec précision centimétrique
- Communication : TCP/IP, RS-485, protocoles propriétaires
- Alimentation : 100-240V AC
- Lecteurs intégrables : RFID, biométrie, QR, facial recognition
- Matériaux : Verre trempé/feuilleté, acier inox, aluminium
- Éclairage : LED RGB pour indication statut
- Modes : Normal, libre passage, sortie libre, bloqué, incendie
- Intégration : BACnet, Modbus, API REST

## Localisation Typique
- Halls d'entrée de tours de bureaux
- Sièges sociaux et corporate
- Centres de données
- Aéroports (zones non publiques)
- Bâtiments gouvernementaux
- Banques et institutions financières
- Hôpitaux (zones restreintes)
