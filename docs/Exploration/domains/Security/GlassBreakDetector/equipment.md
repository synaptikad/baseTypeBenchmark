# Glass Break Detector

## Identifiant
- **Code** : GLASS_BREAK
- **Haystack** : glass-break-sensor
- **Brick** : brick:Glass_Break_Sensor

## Description
Détecteur acoustique spécialisé dans la détection du bris de vitre. Analyse les fréquences sonores caractéristiques produites lors du bris de verre (bruit d'impact, fréquence de cassure). Protège les ouvertures vitrées sans nécessiter de contact physique avec la vitre.

## Fonction
Surveillance acoustique continue des zones vitrées, analyse spectrale des sons, détection du pattern caractéristique du bris de verre, génération d'alarme, protection de multiples vitrages depuis un point central.

## Variantes Courantes
- **Acoustique simple** : Détection par micro uniquement
- **Double technologie** : Acoustique + détection choc/vibration
- **Longue portée** : Jusqu'à 9-12m de rayon
- **Multi-fréquences** : Détecte différents types de verre (trempé, feuilleté, simple)
- **Avec analyse avancée** : Filtrage bruits environnementaux

## Caractéristiques Techniques Typiques
- Portée : 3-12m de rayon
- Zone de couverture : 360° ou 180° selon montage
- Fréquences détectées : 3-5 kHz (impact), 10-15 kHz (cassure)
- Sensibilité réglable : 3-5 niveaux
- Test intégré : Simulateur acoustique
- Communication : Filaire (4-wire), sans fil 868MHz
- Alimentation : 9-16V DC
- Anti-masquage : optionnel sur modèles haut de gamme
- Montage : Plafond ou mural

## Localisation Typique
- Façades vitrées (bureaux, commerces)
- Vitrines
- Salles avec baies vitrées
- Vérandas et atriums
- Portes vitrées
- Fenêtres de locaux sensibles
- Showrooms
