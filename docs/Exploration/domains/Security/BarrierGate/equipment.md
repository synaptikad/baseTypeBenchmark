# Barrier Gate

## Identifiant
- **Code** : BARRIER_GATE
- **Haystack** : barrier-gate
- **Brick** : brick:Vehicle_Barrier

## Description
Barrière automatique pour contrôle d'accès véhicules, composée d'une lisse articulée (bras) se levant et s'abaissant pour autoriser ou bloquer le passage. Utilisée pour parkings, entrées de sites, péages. Peut intégrer lecteur de badges véhicule, boucles de détection, et systèmes anti-crash.

## Fonction
Contrôle accès véhicules, gestion automatique ou manuelle de l'ouverture/fermeture, détection présence véhicule, intégration avec système de gestion parking ou contrôle d'accès, comptage véhicules, sécurité anti-écrasement.

## Variantes Courantes
- **Barrière standard** : Lisse 3-4m, usage modéré
- **Barrière haute cadence** : Ouverture rapide (1-3s), trafic intense
- **Barrière longue portée** : Lisse 5-8m pour larges passages
- **Barrière anti-crash** : Renforcée contre franchissement forcé
- **Barrière escamotable** : Lisse se replie
- **Barrière chaîne** : Pour très larges passages (10m+)

## Caractéristiques Techniques Typiques
- Longueur lisse : 2-8 mètres
- Temps ouverture : 1-6 secondes selon modèle
- Cadence : 50-200 cycles/heure
- Alimentation : 230V AC ou 24V DC
- Motorisation : Électromécanique, hydraulique
- Détection : Boucles magnétiques, cellules photoélectriques
- Sécurité : Anti-écrasement, équilibrage ressort, feu clignotant
- Communication : RS-485, TCP/IP, relais
- Éclairage : Lisse lumineuse LED optionnelle
- Protection : IP44-IP54
- Température : -20°C à +60°C
- Batterie backup : 24-48h autonomie

## Localisation Typique
- Entrées/sorties parkings
- Accès sites industriels
- Postes de garde
- Zones de livraison
- Parkings résidentiels
- Périmètres sécurisés
- Bases de vie

## Relations avec Autres Équipements
- **Alimente** : N/A

- **Contrôlé par** : Parking Management System, Access Controller, Barrier Controller
- **Intègre** : Vehicle Badge Reader (UHF RFID), LPR Camera
- **Reçoit de** : Induction Loop, Photocell, Push Button, Intercom
- **Envoie à** : Parking System, Access Log, Ticket Dispenser
- **Interagit avec** : Traffic Light, Signage, ANPR System

## Quantité Typique par Bâtiment
- Petit (5 étages) : 2-4 barrières (entrée/sortie parking)
- Moyen (15 étages) : 4-8 barrières
- Grand (30+ étages) : 8-20 barrières (multiples parkings/accès)
