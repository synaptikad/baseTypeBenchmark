# Fire Extinguisher Cabinet

## Identifiant
- **Code** : EXT-CAB
- **Haystack** : fire-extinguisher (point de présence)
- **Brick** : N/A

## Description
Armoire de rangement et signalisation des extincteurs portatifs. Dans les smart buildings, peut être équipée de capteurs de présence de l'extincteur pour superviser que les équipements de lutte contre l'incendie sont disponibles à leur emplacement.

## Fonction
Protège les extincteurs des intempéries et dégradations, assure leur visibilité et signalisation réglementaire. Les versions connectées permettent de détecter le retrait d'un extincteur et d'alerter le personnel de sécurité.

## Variantes Courantes
- **Standard non connecté** : Armoire simple avec extincteur
- **Avec contact magnétique** : Détection ouverture porte
- **Avec capteur de présence** : Détection retrait extincteur (RFID, poids)
- **Encastrée** : Dans mur ou cloison
- **En saillie** : Fixée au mur
- **Vitrée** : Visibilité de l'extincteur
- **Tôle pleine** : Protection maximale

## Caractéristiques Techniques Typiques
- Matériau : Tôle acier peinte rouge
- Dimensions : 250x600x180mm à 300x900x250mm
- Capacité : 1-2 extincteurs selon taille
- Signalisation : Pictogramme + texte
- Capteur présence : Contact magnétique, RFID, capteur de poids
- Communication : Contact sec, sans fil (LoRa, Zigbee, BLE)
- Alimentation capteur : Batterie 5-10 ans
- Indice de protection : IP30 (intérieur), IP54 (extérieur)

## Localisation Typique
- Couloirs et circulations
- Halls d'entrée
- Proximité des issues de secours
- Locaux à risques particuliers
- Cuisines (extincteur classe F)
- Locaux électriques (extincteur CO2)
- Parkings
- Distribution : Tous les 15-20 mètres en cheminement

- Petit (5 étages, 2500 m²) : 40-80 unités
- Moyen (15 étages, 15000 m²) : 200-400 unités
- Grand (30+ étages, 50000 m²) : 600-1200 unités
