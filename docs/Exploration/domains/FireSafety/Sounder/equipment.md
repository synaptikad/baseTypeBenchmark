# Sounder / Horn

## Identifiant
- **Code** : SOUNDER
- **Haystack** : fire-alarm-horn
- **Brick** : brick:Fire_Alarm_Horn

## Description
Dispositif sonore d'alarme incendie émettant un signal acoustique puissant et distinctif pour alerter les occupants d'une situation d'urgence. Produit des tonalités spécifiques conformes aux normes (tempo lent ou rapide selon le type d'alarme).

## Fonction
Diffuse l'alarme sonore dans l'ensemble du bâtiment pour alerter les occupants et déclencher l'évacuation. Le signal sonore doit être audible dans toutes les zones, y compris avec bruit ambiant.

## Variantes Courantes
- **Sirène électronique** : Tonalités programmables, plusieurs niveaux sonores
- **Buzzer piézoélectrique** : Signal simple, consommation faible
- **Cloche électromécanique** : Technologie traditionnelle (moins courant)
- **Avec tonalités multiples** : Alarme évacuation / alarme technique
- **Haute puissance** : Pour zones bruyantes (>100 dB)
- **Étanche** : Pour zones humides ou extérieures (IP65-IP67)

## Caractéristiques Techniques Typiques
- Alimentation : 24V DC (via boucle d'alarme)
- Puissance sonore : 85-120 dB à 1 mètre
- Tonalités : Tempo lent (alerte), tempo rapide (évacuation)
- Consommation : 10-50 mA en veille, 50-500 mA en alarme
- Fréquence : 400-2800 Hz
- Indice de protection : IP21-IP67
- Certification : EN 54-3 (Europe), UL 464 (USA)
- Couleur : Rouge normalisé

## Localisation Typique
- Couloirs et circulations
- Halls et espaces communs
- Bureaux ouverts
- Escaliers et cages d'escaliers
- Parkings couverts
- Locaux techniques
- Extérieurs (zones de rassemblement)
- Espaces de vie collectifs

## Relations avec Autres Équipements
- **Alimente** : N/A (dispositif d'alerte)
- **Alimenté par** : Fire Alarm Panel (via boucle sounder)
- **Contrôlé par** : Fire Alarm Panel (FACP/CMSI)
- **Déclenché par** : Smoke Detector, Heat Detector, Manual Call Point
- **Fonctionne avec** : Beacon (alerte visuelle complémentaire)

## Quantité Typique par Bâtiment
- Petit (5 étages, 2500 m²) : 20-40 unités
- Moyen (15 étages, 15000 m²) : 100-200 unités
- Grand (30+ étages, 50000 m²) : 300-600 unités
