# Rainwater Harvesting Tank

## Identifiant
- **Code** : RAINWATER-TANK
- **Haystack** : `rainwater`, `tank`, `equip`
- **Brick** : `brick:Rainwater_Tank`

## Description
Citerne de collecte et stockage des eaux pluviales récupérées depuis les toitures pour réutilisation dans des usages non potables. Système écologique permettant de réduire la consommation d'eau potable et de gérer les eaux de ruissellement.

## Fonction
Collecter les eaux de pluie des toitures via descentes pluviales, les stocker après filtration primaire, et les redistribuer via pompage pour usages non potables (WC, arrosage, nettoyage, process industriels).

## Variantes Courantes
- **Citerne enterrée béton** : Grande capacité, stable température
- **Citerne enterrée polyéthylène** : Installation rapide, étanche
- **Citerne aérienne extérieure** : Accessible, entretien facile
- **Citerne technique sous dalle** : Intégration bâtiment, gain espace

## Caractéristiques Techniques Typiques
- Capacité : 5 m³ à 500 m³ selon surface de toiture collectée
- Matériaux : Béton, PEHD, polyester renforcé
- Filtration amont : Grille feuilles, filtre décantation, trop-plein
- Équipements : Level Sensor, pompe distribution, appoint eau potable
- Protection : Couverture étanche, ventilation, trappe accès
- Traitement optionnel : Filtration fine, UV pour améliorer qualité

## Localisation Typique
- Enterré sous parking ou espaces verts
- Sous-sol technique
- Vide sanitaire
- Extérieur (citerne aérienne)

## Relations avec Autres Équipements
- **Alimente** : WC (chasses), arrosage, Greywater System (appoint), nettoyage
- **Alimenté par** : Collecte toiture, trop-plein vers réseau EP
- **Contrôlé par** : Level Sensor, automate gestion appoint eau potable
- **Associé à** : Potable Water Pump (appoint), Water Filtration System, Water Meter

## Quantité Typique par Bâtiment
- Petit (5 étages) : 0-1 citerne (5-20 m³) si démarche environnementale
- Moyen (15 étages) : 0-1 citerne (30-100 m³) bâtiments certifiés
- Grand (30+ étages) : 1-2 citernes (100-500 m³) grande surface toiture
