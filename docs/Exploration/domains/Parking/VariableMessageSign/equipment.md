# Variable Message Sign

## Identifiant
- **Code** : VMS_PARKING
- **Haystack** : N/A
- **Brick** : N/A

## Description
Panneau à messages variables (VMS) affichant des informations dynamiques textuelles et graphiques pour guidage, alertes, tarifs, ou communications vers usagers du parking. Contenu modifiable à distance en temps réel.

## Fonction
Diffusion d'informations variables vers conducteurs et piétons : disponibilités, tarifs horaires, alertes sécurité, instructions circulation, promotions, messages d'urgence. Améliore expérience utilisateur et communication.

## Variantes Courantes
- **VMS matrice LED** : Texte défilant, graphiques simples
- **VMS LCD couleur** : Contenu riche, images, vidéos
- **VMS pictogrammes** : Symboles directionnels + compteurs
- **VMS extérieur haute luminosité** : Lisibilité plein soleil

## Caractéristiques Techniques Typiques
- Technologie : Matrice LED, LCD/TFT rétroéclairé
- Taille : 40-200 cm (largeur)
- Résolution : Variable selon technologie (8x80 pixels à Full HD)
- Luminosité : 2000-8000 cd/m² (extérieur)
- Protocoles : RS485, Ethernet TCP/IP, HTTP API, SNMP
- Gestion contenu : CMS centralisé, planification messages
- Alimentation : 12-24V DC ou 230V AC
- Température opération : -30°C à +70°C
- Protection : IP54-IP67

## Localisation Typique
- Entrée parking (tarifs, disponibilités)
- Zones de circulation interne (instructions, alertes)
- Zones piétonnes (informations, sécurité)
- Extérieur voirie (pré-information, guidage amont)

## Relations avec Autres Équipements
- **Alimente** : N/A
- **Alimenté par** : Distribution électrique 12-24V DC
- **Contrôlé par** : Parking Management Server, système CMS (Content Management)
- **Interagit avec** : Parking Guidance Display (données complémentaires), système de sécurité (alertes), Parking Revenue Control (tarifs)

## Quantité Typique par Bâtiment
- Petit (parking 50 places) : 1-3 panneaux (entrée + zones clés)
- Moyen (parking 200 places) : 3-8 panneaux (multiples zones info)
- Grand (parking 1000+ places) : 10-20 panneaux (communication étendue)
