# Entry/Exit Station

## Identifiant
- **Code** : ENTRY_EXIT_STATION
- **Haystack** : N/A
- **Brick** : N/A

## Description
Station complète intégrée combinant multiples fonctions d'accès parking : distributeur de tickets, lecteur d'accès, interphone, caméra, et contrôle de barrière. Unité tout-en-un pour gestion entrée ou sortie.

## Fonction
Point de contrôle complet entrée/sortie automatisé. Gère identification véhicules (ticket, badge, plaque), authentification, communication assistance, et commande ouverture barrière. Interface principale usager-système.

## Variantes Courantes
- **Station entrée** : Distributeur ticket + lecteur badge + interphone + caméra
- **Station sortie** : Lecteur ticket + paiement + interphone + caméra
- **Station mixte** : Fonctions entrée et sortie combinées
- **Station compacte** : Version réduite pour voie unique

## Caractéristiques Techniques Typiques
- Intégration : Ticket dispenser, card reader, intercom, ANPR camera
- Écran : Tactile 7-10 pouces, instructions usager
- Lecteurs : RFID, NFC, code-barres, magnétique
- Protocoles : TCP/IP, RS485, Modbus, API REST
- Boîtier : Acier inoxydable ou aluminium, anti-vandalisme
- Alimentation : 230V AC
- Température opération : -30°C à +60°C
- Protection : IP54-IP65

## Localisation Typique
- Entrée principale parking
- Sortie parking
- Accès zones réservées (VIP, abonnés)
- Chaque voie d'accès véhicules
