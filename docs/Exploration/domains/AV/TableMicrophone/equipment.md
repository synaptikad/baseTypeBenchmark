# Table Microphone

## Identifiant
- **Code** : TABLE_MIC
- **Haystack** : N/A
- **Brick** : N/A

## Description
Microphone de conférence individuel posé sur table avec contrôle intégré (mute, LED status). Équipement communicant via Dante, AES67 ou systèmes propriétaires pour contrôle à distance et supervision.

## Fonction
Capture audio individuelle ou par zone pour conférences et réunions. Permet le contrôle par participant (mute local), la supervision centralisée et l'intégration avec systèmes de discussion ou de vote.

## Variantes Courantes
- **Gooseneck Table Microphone** : Col de cygne flexible pour positionnement précis
- **Boundary Table Microphone** : Microphone de surface à zone de capture large
- **Flush-Mount Table Microphone** : Encastré dans table, design discret
- **Discussion System Microphone** : Avec bouton talk/priority, intégration système de discussion
- **Delegate/Chairman Microphone** : Système avec fonctions priorité et contrôle salle

## Caractéristiques Techniques Typiques
- Type : Condensateur électret ou à dos électret
- Directivité : Cardioïde, super-cardioïde, ou hémisphérique
- Portée : 30-80cm typique
- Connectivité : XLR, Dante, proprietary digital bus
- Contrôle : Bouton mute local, LED status (rouge/vert), logiciel distant
- Alimentation : Phantom 48V, PoE, bus system power
- Fréquence : 50Hz-18kHz
- Impédance : 150-200 ohms

## Localisation Typique
- Salles de conseil
- Amphithéâtres et auditoriums
- Salles de conférence formelles
- Centres de formation
- Espaces de délibération

## Relations avec Autres Équipements
- **Alimente** : DSP, Audio Mixer, Discussion System Controller
- **Contrôlé par** : AV Control Processor, Discussion System Controller
- **Connecté via** : Dante Audio Interface, Audio Amplifier
- **Associé à** : Ceiling Speaker, Audio Amplifier
