# Video Scaler

## Identifiant
- **Code** : VIDEO_SCALER
- **Haystack** : N/A
- **Brick** : N/A

## Description
Convertisseur et adaptateur de résolutions vidéo pour compatibilité entre sources et displays. Équipement communicant via IP, RS-232 pour contrôle et monitoring d'état signal.

## Fonction
Conversion de résolutions vidéo (upscaling, downscaling), adaptation de formats (4:3, 16:9, 21:9), désentrelacement et amélioration d'image. Assure compatibilité entre sources variées et displays, élimine les problèmes de synchronisation.

## Variantes Courantes
- **Presentation Scaler** : Scaler simple pour salles de réunion, 1-2 entrées
- **Multi-Format Scaler** : Support formats multiples (HDMI, VGA, DVI, SDI)
- **4K Scaler** : Support entrées/sorties 4K, HDR processing
- **Seamless Scaler** : Commutation sans coupure avec fade/cut effects
- **Windowing Processor** : Multi-images PIP/POP, layouts configurables

## Caractéristiques Techniques Typiques
- Entrées : HDMI, DisplayPort, VGA, DVI, 3G-SDI, USB-C
- Sorties : HDMI, HDBaseT, 3G-SDI
- Résolutions entrée : Jusqu'à 4K60 4:4:4
- Résolutions sortie : Jusqu'à 4K60 4:4:4
- Scaling : Bicubic, bilinear, Lanczos algorithms
- Aspect ratio : Maintain, fill, crop, custom
- Frame rate conversion : Oui sur modèles avancés
- Contrôle : Ethernet (HTTP, Telnet), RS-232, IR
- Latency : <50ms typique
- HDR : HDR10, Dolby Vision (modèles premium)

## Localisation Typique
