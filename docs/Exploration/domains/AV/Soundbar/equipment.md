# Soundbar

## Identifiant
- **Code** : SOUNDBAR
- **Haystack** : N/A
- **Brick** : N/A

## Description
Système audio tout-en-un combinant haut-parleurs et souvent microphones dans un format barre unique. Équipement communicant via USB, Bluetooth, IP pour contrôle et intégration avec codecs vidéo.

## Fonction
Solution audio simple et intégrée pour petites salles de réunion et huddle rooms. Combine diffusion audio (stereo/spatial) et capture microphone dans un seul appareil. Intégration plug-and-play avec systèmes de visioconférence.

## Variantes Courantes
- **USB Soundbar** : Connexion USB simple pour BYOD, avec microphones intégrés
- **Bluetooth Soundbar** : Connectivité wireless pour flexibilité
- **Video Soundbar** : Intègre caméra + audio + microphones (all-in-one)
- **Dante Soundbar** : Connectivité Dante pour infrastructure AV professionnelle
- **AI Soundbar** : Beamforming microphones et traitement audio intelligent

## Caractéristiques Techniques Typiques
- Configuration audio : 2.0, 2.1 (avec subwoofer), 3.0 stereo
- Puissance : 20W à 80W total
- Microphones : 4-8 microphones array avec beamforming
- Portée microphone : 3-5m typique
- Connectivité : USB 3.0, USB-C, HDMI, Bluetooth, Ethernet
- Contrôle : USB HID, HTTP API, Bluetooth control
- Traitement : AEC, AGC, noise suppression
- Compatibilité : Zoom, Teams, Meet certified (certains modèles)

## Localisation Typique
- Huddle rooms (2-6 personnes)
- Focus rooms individuelles
- Petites salles de réunion
- Home offices executives
- Espaces collaboration informels

## Relations avec Autres Équipements
- **Alimente** : Display (connexion HDMI ARC/eARC)
- **Alimenté par** : Video Conference Codec, PC/Laptop (USB)
- **Contrôlé par** : Video Conference Codec, Touch Panel Controller
- **Intégré avec** : Fixed Camera (modèles all-in-one)
