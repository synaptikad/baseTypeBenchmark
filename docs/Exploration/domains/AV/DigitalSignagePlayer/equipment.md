# Digital Signage Player

## Identifiant
- **Code** : DS_PLAYER
- **Haystack** : N/A
- **Brick** : N/A

## Description
Lecteur média dédié pour affichage dynamique et signalétique numérique. Équipement communicant via IP pour gestion de contenu à distance, scheduling et monitoring d'état.

## Fonction
Diffusion de contenu multimédia (vidéos, images, web, data) sur displays pour communication corporate, wayfinding, information temps réel. Gestion centralisée du contenu, scheduling par horaires/événements, monitoring de l'état player et display.

## Variantes Courantes
- **System-on-Chip Player** : Player intégré dans le display (SoC)
- **Media Player Appliance** : Box dédié externe, Linux ou Android
- **PC-Based Player** : Mini-PC Windows/Linux pour contenu complexe
- **Cloud Player** : Player léger avec streaming cloud du contenu
- **4K/8K Player** : Support contenu ultra-haute résolution

## Caractéristiques Techniques Typiques
- OS : Android, Linux, Windows 10 IoT, Chrome OS
- CPU : ARM Cortex, Intel x86 selon performances
- RAM : 2GB à 16GB
- Storage : 8GB à 256GB SSD
- Résolution : 1080p à 8K selon modèle
- Connectivité : HDMI, DisplayPort, Ethernet, WiFi
- Contrôle : HTTP API, SNMP, proprietary CMS protocols
- CMS : Content Management System cloud ou on-premise
- Formats : MP4, MOV, JPG, PNG, HTML5, RSS, IPTV streams
- Monitoring : Screen on/off detection, playback status, health check

## Localisation Typique
- Halls d'entrée et lobbies
- Couloirs et circulations
- Ascenseurs (écrans d'information)
- Cafétérias et espaces communs
- Salles d'attente
- Façades extérieures (outdoor)

## Relations avec Autres Équipements
- **Alimente** : Display, LED Video Wall Panel, Projector (rare)
- **Contrôlé par** : Digital Signage CMS (cloud/on-premise), AV Control Processor
- **Connecté via** : AV Matrix Switcher (certains cas)

## Quantité Typique par Bâtiment
- Petit (5 étages) : 8-20 players
- Moyen (15 étages) : 30-80 players
- Grand (30+ étages) : 100-250 players
