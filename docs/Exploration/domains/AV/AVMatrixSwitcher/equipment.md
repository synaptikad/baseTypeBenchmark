# AV Matrix Switcher

## Identifiant
- **Code** : AV_MATRIX
- **Haystack** : N/A
- **Brick** : N/A

## Description
Commutateur matriciel pour routage de signaux audio et vidéo entre sources multiples et destinations multiples. Équipement communicant via IP, RS-232 pour contrôle automatisé et supervision d'état.

## Fonction
Routage flexible de sources vidéo (codecs, PCs, lecteurs media) vers displays et projecteurs. Permet le contrôle centralisé, les présets de routage, le scaling vidéo et l'intégration avec systèmes de contrôle AV. Point central de l'infrastructure vidéo.

## Variantes Courantes
- **HDMI Matrix Switcher** : Routage HDMI, 4x4 à 32x32
- **HDBaseT Matrix Switcher** : Distribution longue distance (100m) avec PoE
- **Modular Matrix Switcher** : Châssis modulaire avec cartes I/O mixtes (HDMI, HDBaseT, SDI, DVI)
- **4K/8K Matrix Switcher** : Support résolutions ultra-hautes 4K60 4:4:4, 8K
- **AV over IP Matrix** : Routage sur réseau Ethernet avec encodeurs/décodeurs
- **Seamless Switcher** : Commutation sans coupure (frame-accurate)

## Caractéristiques Techniques Typiques
- Taille matrice : 4x4 à 144x144 (modular systems)
- Résolutions : 1080p60, 4K30, 4K60, 8K30
- Signaux : HDMI 2.0/2.1, HDBaseT, 3G-SDI, DVI
- Distance : 10m (HDMI), 100m (HDBaseT), illimité (AV over IP)
- HDCP : Support HDCP 1.4, 2.2, 2.3
- Contrôle : Ethernet (HTTP API, Telnet), RS-232, IR, GPIO
- Audio : De-embedding/embedding, routing séparé audio
- Scaling : Intégré ou via modules séparés
- Latency : <1 frame (seamless), <100ms (AV over IP)
