# Media Server

- **Code** : MEDIA_SRV
- **Haystack** : N/A

## Description
Serveur de lecture média haute performance pour contenu vidéo complexe, multi-écrans et temps réel. Équipement communicant via IP pour contrôle de playlists, synchronisation et monitoring.

## Fonction
Diffusion de contenu vidéo synchronisé multi-écrans pour événements, présentations corporate, show rooms, environnements immersifs. Gestion de contenu haute résolution (4K-8K), effets temps réel, mapping de projection, synchronisation frame-accurate. Utilisé pour applications professionnelles critiques.

## Variantes Courantes
- **Presentation Media Server** : Lecture contenu PowerPoint, Keynote avec effets
- **Video Playback Server** : High-bitrate video playback, multi-output sync
- **Interactive Media Server** : Support contenu interactif et génératif
- **Projection Mapping Server** : Warping, blending, mapping géométrique
- **GPU Render Server** : Real-time rendering 3D pour visualisation

## Caractéristiques Techniques Typiques
- GPU : NVIDIA RTX série professional, multi-GPU
- Sorties : 4 à 16 outputs DisplayPort/HDMI
- Résolution : 4K60 par output, 8K support
- Codecs : H.264, H.265, HAP, ProRes, DNxHD, image sequences
- Contrôle : OSC, MIDI, DMX, HTTP API, Art-Net
- Synchronisation : Genlock, timecode (LTC, MTC), frame-accurate
- Interactivité : Touch, sensors, Kinect, external control
- Effets : Warping, blending, edge blending pour multi-proj
- Layers : Multiple video layers avec compositing temps réel
- Playout : Timeline-based ou triggered playout
