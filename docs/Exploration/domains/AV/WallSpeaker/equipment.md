# Wall Speaker

## Identifiant
- **Code** : WALL_SPK
- **Haystack** : N/A
- **Brick** : N/A

## Description
Haut-parleur monté en surface ou encastré dans les murs pour diffusion audio directionnelle. Peut être communicant via Dante ou contrôlé via amplificateurs DSP.

## Fonction
Diffusion audio pour espaces nécessitant couverture directionnelle ou renforcement local. Utilisé pour sonorisation périphérique, évacuation, ou audio d'ambiance. Modèles réseau permettent contrôle de volume par zone.

## Variantes Courantes
- **Surface Mount Wall Speaker** : Montage saillie, installation simple
- **In-Wall Speaker** : Encastré dans mur, design discret
- **Column Speaker** : Format colonne pour grande portée et directivité
- **Pendant Speaker** : Suspendu, pour zones hautes ou atriums
- **Outdoor Wall Speaker** : Étanche IP66/IP67 pour usage extérieur

## Caractéristiques Techniques Typiques
- Configuration : 2-way, 3-way (modèles premium)
- Puissance : 10W à 100W
- SPL max : 90-105dB @ 1m
- Bande passante : 50Hz-20kHz
- Ligne : 70V/100V ou low-impedance
- Directivité : 90° x 90° à 40° x 20° (column)
- Protection : IP rating pour extérieur
- Matériaux : ABS, métal, weather-resistant coating

## Localisation Typique
- Couloirs et circulations
- Espaces extérieurs et terrasses
- Parkings couverts
- Zones industrielles du bâtiment
- Escaliers et cages d'escalier
- Zones d'évacuation
