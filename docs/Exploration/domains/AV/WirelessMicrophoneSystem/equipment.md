# Wireless Microphone System

## Identifiant
- **Code** : WIRELESS_MIC
- **Haystack** : N/A
- **Brick** : N/A

## Description
Système de microphones sans fil avec récepteurs, émetteurs et gestion spectrale RF. Équipement communicant via Ethernet pour coordination fréquentielle, monitoring batterie et contrôle à distance.

## Fonction
Capture audio mobile pour présentateurs, événements et espaces flexibles. Permet la mobilité sans câbles, la gestion automatique des fréquences RF, le monitoring de l'état (batterie, RF quality) et l'intégration avec systèmes AV.

## Variantes Courantes
- **Handheld Wireless Microphone** : Micro main traditionnel pour présentations
- **Bodypack Wireless System** : Émetteur ceinture avec lavalier ou headset
- **Desktop Wireless Boundary** : Microphone de table sans fil
- **Digital Wireless System** : Transmission numérique cryptée, latence ultra-faible
- **Rechargeable System** : Système avec stations de charge et batteries lithium-ion

## Caractéristiques Techniques Typiques
- Bandes fréquences : UHF 470-698 MHz (selon région), 2.4GHz digital, 5.8GHz
- Portée : 30-100m selon environnement et puissance
- Canaux simultanés : 4 à 32+ par système
- Contrôle : Ethernet (HTTP API, SNMP), IR sync
- Monitoring : RF level, battery status, audio level, frequency coordination
- Latence : <3ms (analog), <4ms (digital)
- Encryption : AES-256 pour systèmes digitaux
- Auto-frequency scan et coordination
- Network control via software

## Localisation Typique
- Auditoriums et amphithéâtres
- Salles polyvalentes
- Espaces événementiels
- Grandes salles de formation
- Studios internes

## Relations avec Autres Équipements
- **Alimente** : DSP, Audio Mixer, Audio Amplifier
- **Contrôlé par** : AV Control Processor
- **Connecté via** : Dante Audio Interface (certains modèles), Audio Distribution Amplifier
