# Projector

## Identifiant
- **Code** : PROJECTOR
- **Haystack** : N/A
- **Brick** : N/A

## Description
Dispositif de projection vidéo pour affichage grand format dans les environnements professionnels. Équipement communicant via IP, RS-232 ou HDBaseT pour supervision de l'état (lampe, température) et contrôle à distance.

## Fonction
Projection de contenu vidéo pour présentations, formations, visioconférence ou affichage événementiel. Permet le contrôle centralisé (ON/OFF, source, zoom, keystone) et monitoring de l'état opérationnel (heures lampe, température).

## Variantes Courantes
- **DLP Projector** : Technologie à miroirs micro-mécaniques, haute luminosité
- **LCD Projector** : Technologie à cristaux liquides, reproduction couleur fidèle
- **Laser Projector** : Source laser au lieu de lampe, maintenance réduite, durée de vie 20000h+
- **Short-Throw Projector** : Distance de projection réduite (0.5-1.5m)
- **Ultra Short-Throw Projector** : Distance extrêmement courte (0.2-0.5m)
- **Installation Projector** : Haute luminosité pour grandes salles, avec shift optique

## Caractéristiques Techniques Typiques
- Luminosité : 3000-7000 ANSI lumens (salles de réunion), 7000-30000 lumens (auditoriums)
- Résolutions : WXGA (1280x800), Full HD (1920x1080), WUXGA (1920x1200), 4K UHD
- Throw ratio : 0.2:1 (ultra short) à 2.5:1 (long throw)
- Connectivité : HDMI, HDBaseT, 3G-SDI, USB-C
- Contrôle : RS-232, RJ45 (IP control), IR
- Protocoles : PJLink (standard industrie), AMX DXLink, proprietary IP protocols
- Source lumineuse : Lampe UHP/UHLD (2000-5000h), LED, Laser (20000h+)

## Localisation Typique
