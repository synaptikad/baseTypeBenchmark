# Wireless Presentation Gateway

## Identifiant
- **Code** : WIRELESS_PRES
- **Haystack** : N/A
- **Brick** : N/A

## Description
Système de partage d'écran sans fil permettant aux utilisateurs de projeter depuis leurs appareils personnels. Équipement communicant via IP pour gestion, monitoring et intégration avec systèmes de contrôle.

## Fonction
Projection sans fil depuis laptops, tablets, smartphones vers displays de salle. Élimine les câbles HDMI, supporte multi-utilisateurs, modération de contenu, annotation collaborative. Intégration avec codecs de visioconférence pour content sharing. Gestion centralisée et analytics d'usage.

## Variantes Courantes
- **Hardware Gateway** : Box dédiée HDMI, support Miracast/AirPlay/Chromecast
- **Software-Based Gateway** : Application uniquement, pas de hardware propriétaire
- **USB-C Gateway** : Connexion filaire USB-C en fallback du wireless
- **4K Wireless Gateway** : Support streaming 4K60
- **Enterprise Gateway** : Gestion centralisée, authentication, encryption

## Caractéristiques Techniques Typiques
- Résolution : 1080p60, 4K30 (premium 4K60)
- Protocoles : Miracast, AirPlay, Chromecast, proprietary
- Connectivité : WiFi 5/6, Ethernet, HDMI output
- Clients : Windows, macOS, iOS, Android, Chrome OS
- Latency : 50-200ms selon protocole
- Multi-user : 4-16 participants simultanés (split screen)
- Modération : Host approval, PIN code access
- Contrôle : HTTP API, SNMP, cloud management portal
- Intégration : HDMI input pour passthrough, USB pour codec integration
- Sécurité : WPA2/WPA3, AES encryption, network isolation
