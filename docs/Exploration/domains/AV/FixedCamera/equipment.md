# Fixed Camera

## Identifiant
- **Code** : FIXED_CAMERA
- **Haystack** : N/A
- **Brick** : N/A

## Description
Caméra à position fixe pour visioconférence et capture vidéo. Équipement communicant via USB, IP ou HDMI avec contrôle logiciel pour paramètres image (exposition, balance blancs, cadrage numérique).

## Fonction
Capture vidéo haute qualité pour visioconférence dans les petites salles et espaces personnels. Solution simple, plug-and-play, souvent avec champ de vision large et cadrage automatique par IA.

## Variantes Courantes
- **Wide-Angle Conference Camera** : Champ de vision 90-120° pour petites salles
- **Ultra Wide-Angle Camera** : FOV 180° pour salles rectangulaires
- **AI Auto-Framing Camera** : Cadrage automatique intelligent des participants
- **Soundbar Integrated Camera** : Caméra intégrée dans une soundbar tout-en-un
- **USB Conference Camera** : Connexion USB 3.0 simple pour BYOD

## Caractéristiques Techniques Typiques
- Résolutions : 1080p30/60, 4K30
- Field of View : 80° à 180° selon modèle
- Connectivité : USB 3.0, HDMI, USB-C, IP (RTSP)
- Contrôle : UVC (USB Video Class), HTTP API, proprietary software
- Auto-framing : IA-based participant detection
- Audio : Souvent intégré (microphone array)
- Montage : Table, écran, mur, plafond

## Localisation Typique
- Huddle rooms (2-4 personnes)
- Focus rooms individuelles
- Petites salles de réunion
- Home offices executives
- Training rooms

## Relations avec Autres Équipements
- **Alimente** : Video Conference Codec, Display (via USB)
- **Contrôlé par** : Video Conference Codec, PC/Laptop (BYOD)
- **Intégré avec** : Soundbar, Microphone Array
