# DSP (Digital Signal Processor)

## Identifiant
- **Code** : DSP
- **Haystack** : N/A
- **Brick** : N/A

## Description
Processeur de signal audio numérique pour traitement, routage et gestion audio avancée. Équipement hautement communicant via Ethernet pour configuration, contrôle en temps réel et monitoring audio.

## Fonction
Centre névralgique du traitement audio : égalisation, compression, limiteur, délais, routage matriciel, suppression d'écho, gain automatique. Point central de contrôle et monitoring pour systèmes audio complexes. Intégration avec codecs, amplificateurs et contrôleurs AV.

## Variantes Courantes
- **Room DSP** : DSP pour salle unique, 4-8 entrées/sorties
- **Conferencing DSP** : Optimisé pour AEC et traitement visioconférence
- **Networked DSP** : DSP Dante/AVB avec routage audio over IP
- **Large-Scale DSP** : Processeur 16+ canaux pour systèmes complexes
- **Modular DSP** : Châssis modulaire avec cartes I/O configurables

## Caractéristiques Techniques Typiques
- Entrées : 4 à 64+ canaux (microphones, line, Dante, AES67)
- Sorties : 4 à 64+ canaux (speakers, Dante, AES67)
- Traitement : EQ parametric, compressor, limiter, gate, delay, crossover, AEC
- Latence : <3ms typique
- Échantillonnage : 48kHz/96kHz, 24-bit
- Contrôle : Ethernet (HTTP API, Telnet, proprietary), RS-232, GPIO
- Présets : Stockage configurations multiples
- Dante/AES67 : Intégration native pour audio over IP
- Monitoring : VU meters, signal presence, fault detection

## Localisation Typique
- Salles de visioconférence (rack)
- Auditoriums et amphithéâtres
- Salles de conseil
- Locaux techniques AV centraux
- Multi-purpose rooms
