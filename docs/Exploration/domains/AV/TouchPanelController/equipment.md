# Touch Panel Controller

## Identifiant
- **Code** : TOUCH_PANEL
- **Haystack** : N/A
- **Brick** : N/A

## Description
Interface utilisateur tactile pour contrôle de systèmes AV. Équipement communicant via IP, WiFi pour envoi de commandes au AV Control Processor et affichage d'états.

## Fonction
Interface de contrôle utilisateur pour salles AV : sélection de sources, contrôle volume, présets de configuration, commande d'éclairage et stores. Point d'interaction principal pour utilisateurs non-techniques. Feedback visuel de l'état système.

## Variantes Courantes
- **Wall-Mount Touch Panel** : Panel fixe encastré ou en saillie, 5"-10"
- **Tabletop Touch Panel** : Panel de table avec dock, orientable
- **Wireless Touch Panel** : Tablette WiFi avec app de contrôle
- **Room Scheduling Touch Panel** : Dual-function : scheduling + room control
- **Button Panel** : Panel simplifié avec boutons physiques pour actions basiques

## Caractéristiques Techniques Typiques
- Taille écran : 5" à 15.6" selon application
- Résolution : 720p à 1080p
- Connectivité : Ethernet PoE, WiFi, Bluetooth
- Technologie : Capacitif multi-touch
- Contrôle : HTTP, WebSocket, proprietary protocols
- Programmation : Interface designer drag-and-drop
- Feedback : État temps réel des équipements
- Sécurité : PIN code, authentication
- Montage : VESA, custom bracket, flush mount, table dock
