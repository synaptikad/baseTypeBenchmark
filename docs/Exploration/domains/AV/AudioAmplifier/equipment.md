# Audio Amplifier

## Identifiant
- **Code** : AUDIO_AMP
- **Haystack** : N/A
- **Brick** : N/A

## Description
Amplificateur de puissance audio pour alimentation de haut-parleurs passifs. Équipement communicant via Ethernet, RS-232 pour monitoring température, protection, charge et contrôle à distance.

## Fonction
Amplification du signal audio vers les haut-parleurs passifs dans les installations distribuées. Permet le monitoring d'état (température, load, protection), contrôle de volume par zone et intégration avec systèmes de gestion AV.

## Variantes Courantes
- **Multi-Channel Amplifier** : 2 à 16 canaux indépendants
- **Distributed Audio Amplifier** : 70V/100V pour lignes distribuées commerciales
- **DSP Amplifier** : Amplificateur avec DSP intégré (EQ, delay, crossover)
- **Dante Amplifier** : Entrées Dante pour infrastructure audio réseau
- **Class-D Amplifier** : Haute efficacité énergétique (>90%), compact

## Caractéristiques Techniques Typiques
- Canaux : 1 à 16 sorties
- Puissance par canal : 50W à 500W @ 8 ohms
- Sorties : Low-impedance (4/8 ohms), 70V/100V line
- Entrées : XLR, Phoenix, Dante, AES67, analog balanced
- Contrôle : Ethernet (HTTP, SNMP), RS-232, GPIO
- Monitoring : Température, load impedance, protection status, signal presence
- Protection : Short circuit, thermal, DC, overcurrent
- Efficacité : 70% (Class AB) à 95% (Class D)
- Refroidissement : Convection forcée ou passive

## Localisation Typique
- Locaux techniques AV (rack rooms)
- Salles serveurs périphériques
- Faux-plafonds (modèles compacts)
- Armoires de brassage par étage
