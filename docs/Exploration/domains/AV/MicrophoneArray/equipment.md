# Microphone Array

## Identifiant
- **Code** : MIC_ARRAY
- **Haystack** : N/A
- **Brick** : N/A

## Description
Système multi-microphones intégré pour capture audio directionnelle avec traitement numérique. Équipement communicant via USB, Dante, ou AES67 avec contrôle logiciel pour beamforming et suppression de bruit.

## Fonction
Capture audio intelligente pour visioconférence avec formation de faisceaux (beamforming), suppression d'écho et de bruit, détection automatique des locuteurs. Assure une qualité audio optimale sans microphones individuels.

## Variantes Courantes
- **Ceiling Microphone Array** : Montage plafond, capture de zone 3-6m radius
- **Table Microphone Array** : Posé sur table, capture 360° ou directionnel
- **Linear Microphone Array** : Barre linéaire pour grande table de conférence
- **Pendant Microphone Array** : Suspendu au-dessus de la table
- **In-Ceiling Beamforming Array** : Système encastré avec multiple zones de couverture

## Caractéristiques Techniques Typiques
- Nombre de microphones : 4 à 32 capsules selon modèle
- Portée de capture : 3m à 10m selon technologie
- Technologie : Beamforming adaptatif, AEC (Acoustic Echo Cancellation), AGC (Auto Gain Control)
- Connectivité : USB, Dante, AES67, AES3, analog balanced
- Contrôle : Proprietary software, HTTP API, Dante Controller
- Zones de couverture : 1 à 8 zones beam steering
- Fréquence : 80Hz-20kHz
- SNR : 70-80dB typique
- PoE : PoE+ pour modèles réseau

## Localisation Typique
- Salles de visioconférence
- Salles de réunion moyennes à grandes
- Salles de conseil
- Training rooms
- Espaces collaboration ouverts
