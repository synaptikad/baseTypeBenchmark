# PTZ Camera

## Identifiant
- **Code** : PTZ_CAMERA
- **Haystack** : N/A
- **Brick** : N/A

## Description
Caméra motorisée à panoramique, inclinaison et zoom (Pan-Tilt-Zoom) pour visioconférence et surveillance. Équipement communicant via IP, RS-232/485, VISCA pour contrôle précis et présets de position.

## Fonction
Capture vidéo haute qualité avec cadrage dynamique pour visioconférence. Permet le suivi automatique des participants, les présets de position, et le contrôle à distance. Intégration avec codecs vidéo et systèmes de contrôle AV.

## Variantes Courantes
- **Conference PTZ Camera** : Optimisée pour visioconférence, 1080p60 ou 4K30
- **Presentation Tracking Camera** : Suivi automatique du présentateur via AI
- **Ceiling Mount PTZ** : Montage plafond pour vue plongeante
- **Auto-Tracking PTZ** : Suivi automatique multi-participants avec IA
- **ePTZ Camera** : Caméra fixe 4K/8K avec zoom numérique multiple sorties

## Caractéristiques Techniques Typiques
- Résolutions : 1080p60, 4K30, 4K60 (haut de gamme)
- Zoom optique : 3x à 30x selon modèle
- Pan range : +/- 170° à 340° total
- Tilt range : +90° / -30° typique
- Vitesse : 0.1° à 300°/sec
- Connectivité : HDMI, 3G-SDI, USB 3.0, NDI, IP streaming (RTSP/RTMP)
- Contrôle : VISCA over IP/RS-232/RS-422, HTTP API, ONVIF
- Présets : 10 à 255 positions mémorisées
- Audio : Optionnel, embedded audio input
- PoE : PoE+ (802.3at) pour modèles IP
