# Display

## Identifiant
- **Code** : DISPLAY
- **Haystack** : N/A
- **Brick** : N/A

## Description
Écran plat numérique utilisé pour l'affichage de contenu audiovisuel dans les environnements commerciaux. Équipement communicant via IP, RS-232 ou HDBaseT pour supervision et contrôle à distance.

## Fonction
Affichage de contenu vidéo pour visioconférence, présentations, signalétique numérique ou collaboration. Permet le contrôle centralisé (ON/OFF, volume, source, luminosité) et la supervision d'état.

## Variantes Courantes
- **Professional Display** : Écran commercial conçu pour fonctionnement continu 16/7 ou 24/7
- **Interactive Flat Panel** : Écran tactile interactif pour collaboration
- **Video Wall Display** : Écran optimisé pour assemblage en mur d'images avec bordures ultra-fines
- **Outdoor Display** : Écran durci pour usage extérieur avec haute luminosité

## Caractéristiques Techniques Typiques
- Tailles courantes : 32" à 98" (commercial), jusqu'à 110" pour usage spécialisé
- Résolutions : Full HD (1920x1080), 4K UHD (3840x2160), 8K pour vidéo wall
- Luminosité : 350-500 cd/m² (indoor), 2500+ cd/m² (outdoor)
- Connectivité : HDMI, DisplayPort, HDBaseT, USB-C
- Contrôle : RS-232, RJ45 (IP control), CEC, IR
- Protocoles : PJLink, proprietary IP protocols, SNMP
- Audio intégré : Haut-parleurs 10-20W (optionnel)

## Localisation Typique
- Salles de réunion et visioconférence
- Espaces de collaboration
- Halls d'entrée (signalétique)
- Salles de formation
- Espaces d'accueil
- Couloirs (wayfinding)

## Relations avec Autres Équipements
- **Alimenté par** : AV Matrix Switcher, Video Conference Codec, Digital Signage Player, Wireless Presentation Gateway
- **Contrôlé par** : AV Control Processor, Touch Panel Controller, Room Scheduling Display
- **Connecté via** : HDBaseT Extender, HDMI Extender

## Quantité Typique par Bâtiment
- Petit (5 étages) : 15-30 displays
- Moyen (15 étages) : 80-150 displays
- Grand (30+ étages) : 200-400 displays
