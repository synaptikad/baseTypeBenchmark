# Stair Lift

## Identifiant
- **Code** : STAIR_LIFT
- **Haystack** : stairLift + equip
- **Brick** : N/A (équipement rarement intégré BMS)

## Description
Monte-escalier suivant la rampe d'un escalier existant pour le transport d'une personne assise. Équipement d'accessibilité PMR principalement résidentiel, parfois présent dans petits bâtiments tertiaires.

## Fonction
Permettre aux personnes à mobilité réduite de franchir des escaliers en position assise. Solution d'accessibilité pour bâtiments existants sans espace pour ascenseur.

## Variantes Courantes
- **Monte-escalier droit** : Escalier rectiligne simple
- **Monte-escalier tournant** : Rail courbé, escaliers complexes
- **Monte-escalier extérieur** : Protection météo
- **Monte-escalier debout** : Position debout pour espaces réduits
- **Monte-escalier double rail** : Pour escaliers larges

## Caractéristiques Techniques Typiques
- Capacité : 120 kg à 160 kg (1 personne)
- Vitesse : 0.1 m/s à 0.15 m/s
- Rail fixé sur marches ou mur
- Siège pliable avec ceinture sécurité
- Alimentation : Batterie rechargeable 24V
- Commande : Joystick sur accoudoir + télécommande
- Intégration BMS : Très rare (usage autonome)

## Localisation Typique
- Escaliers résidentiels
- Petits immeubles de bureaux
- Établissements recevant du public (ERP) petite taille
- Églises et bâtiments historiques
- Maisons de retraite

## Relations avec Autres Équipements
- **Alimente** : N/A
- **Alimenté par** : Electrical Outlet (chargeur batterie)
- **Contrôlé par** : Standalone Controller (autonome)
- **Supervise par** : Généralement non supervisé
- **Interagit avec** : Emergency Call System (optionnel)

## Quantité Typique par Bâtiment
- Petit (5 étages) : 0-1 monte-escalier (si bâtiment ancien sans ascenseur)
- Moyen (15 étages) : 0 monte-escalier (présence obligatoire d'ascenseurs)
- Grand (30+ étages) : 0 monte-escalier (non applicable)
