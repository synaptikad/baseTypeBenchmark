# Requêtes cibles

Huit requêtes conceptuelles sont prévues pour couvrir les parcours graphes et la partie séries temporelles :
1. Chaîne énergétique profondeur ≤10
2. Impact AHU -> zones -> espaces -> points profondeur ≤8
3. Serve space (quels équipements servent un espace donné)
4. Inventaire des points par quantité/unité sur un étage
5. Détection des orphelins
6. Séries temporelles avec agrégation horaire
7. Détection de dérive (top 20)
8. Parcours mixte tenant -> served -> timeseries
