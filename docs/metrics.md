# Métriques

Les métriques principales retenues :
- p50 et p95 des latences de requêtes
- QPS (queries par seconde)
- time_to_ingest (temps d'ingestion)
- RAM steady-state et RAM peak par conteneur
- disk usage des volumes
- CPU (optionnel selon disponibilité des mesures)

Les résultats seront exportés en JSON et CSV pour faciliter l'analyse et la comparaison.
