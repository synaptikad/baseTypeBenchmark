# Génération de dataset

Ce dossier hébergera les scripts de génération de données synthétiques. Les objectifs clés :
- Produire des graphes et séries temporelles cohérents avec un seed explicite.
- Permettre deux profils de volumétrie (laptop et server) en configurant le SCALE_MODE.
- Émettre les données dans des formats adaptés aux trois moteurs (SQL, CSV, JSON-LD).

Aucun script n'est fourni pour l'instant ; ils seront ajoutés dans une itération ultérieure.
